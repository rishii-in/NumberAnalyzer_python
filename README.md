# 🔢 Number Analyzer (Python Project)

An interactive **menu-driven Python application** that performs multiple number-based operations using loops and basic logic.

---

## 🚀 Features

* 🔁 Reverse a Number
* ➕ Sum of Digits
* 🔍 Prime Number Check
* ✖️ Multiplication Table Generator
* ⭐ Star Pattern Printing
* ❌ Exit Option

---

## 🧠 Concepts Used

This project is built to strengthen core Python fundamentals:

* while loop (menu-driven program)
* for loop (iteration & patterns)
* nested loops (pattern printing)
* conditional statements (if-elif-else)
* basic number logic

---

## 💻 How It Works

* The program runs in a loop and displays a menu.
* User selects an option (1–6).
* Based on the input, the corresponding operation is performed.
* The program continues until the user chooses to exit.
